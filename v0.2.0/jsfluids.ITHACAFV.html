(async) loadMesh(mesh)

Description:

  • Loads a VTK Unstructured Grid (.vtu) mesh from either an URL or a buffer.

    This function initialize the module and the sence, and sets the number of cells.

See:
  • https://play.simzero.com/#1KEMHY#1
Example
model.loadMesh(meshURL).then(() => {
  ...
});
Parameters:
Name Type Description
mesh string | Buffer

The mesh to load, which can be either an URL or a buffer of a VTU file:

  • If mesh is a string, it is treated as an URL and the mesh will be loaded from the URL.
  • If mesh is a buffer, the mesh will be loaded from the buffer.

(async) loadModel(mesh)

Description:

  • Loads an ITHACA-FV model with its matrices and related files bundled in a ZIP files.

    This function initialize the module and the scene, and sets the number of cells.

See:
  • https://play.simzero.com/#1KEMHY#1
Example
model.loadModel(romURL).then(() => {
  ...
});
Parameters:
Name Type Description
mesh string | Buffer

The model to load, which can be either an URL or a buffer of an ITHACA-FV ZIP file:

  • If mesh is a string, it is treated as an URL and the model will be loaded from the URL.
  • If mesh is a buffer, the mesh will be loaded from the buffer.

setComponent(dict) → {string}

Description:

  • Sets a visualization component and returns a string representation of a GLTF

See:
  • https://play.simzero.com/#1KEMHY#1
Properties:
Name Type Attributes Description
dict.component string

The component to visualize. Must be one of "surface", "plane", or "streamlines".
dict.properties.origin Array.<number> <optional>

Required when component is "plane. The origin vector, which is an array with three numbers representing the x,y,z coordinates.
dict.properties.normal Array.<number> <optional>

Required when component is "plane. The normal vector, which is an array with three numbers representing the x,y,z coordinates.
dict.properties.field string <optional>

Required when component is "streamlines". The field to calculate the streamlines.
dict.properties.center Array.<number> <optional>

Required when component is "streamlines. The center of the streamlines seeding sphere, which is an array with three numbers representing the x,y,z coordinates.
dict.properties.radius number <optional>

Required when component is "streamlines". The radius of the streamlines seeding sphere.
dict.properties.propagation number <optional>

Required when component is "streamlines". The propagation lenght of the streamlines.
dict.properties.tubeRadius number <optional>

Required when component is "streamlines". The radius of the streamlines tube.
dict.properties.tubeSides number <optional>

Required when component is "streamlines".The number of sides for streamlines tubes.
dict.properties.resolution number <optional>

Required when component is "streamlines". The numbers of streamlines to generate.
Example
var gltfSurfce = model.setComponent({
  type: "surface"
});

var gltfPlane = model.setComponent({
  type: "plane",
  properties: {
    origin: [0, 0, 0],
    normal: [1, 0, 0]
  }
});

var gltfStreamlines = model.setComponent({
  type: "streamlines",
  properties: {
    field: "U",
    center: [0, 0, 0],
    radius: 1.0,
    resolution: 8,
    tubeRadius: 0.1,
    tubeSides: 30,
    propagation: 300.0
   }
 });
Parameters:
Name Type Description
dict Object
Returns:

The GLTF data as a string

Type
string
